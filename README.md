# Web-page-look
Simple webpage created using HTML to learn the basic structure and content organisation 
🌐 First Web Page (HTML)

 📌 Description
This project is a simple **first web page** created using HTML. It demonstrates the basic structure of a web page including headings, paragraphs, images, and links.

The main goal of this project is to understand how content is structured and displayed using HTML.

🚀 Features
- Headings and paragraphs
- Image display
- Hyperlinks
- Basic page structure

- Understand webpage structure  
- Practice content organization
